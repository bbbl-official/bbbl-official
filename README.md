## QUICK DEPLOY

</div>

<div align="center">

**1.** Get the worker
```
git clone https://github.com/bbbl-official/bbbl-gateway
```

**2.** Install Wrangler
```
npm install -g wrangler
```

**3.** Login to Cloudflare
```
wrangler login
```

**4.** Deploy (free tier works)
```
wrangler deploy
```

**5.** Done! Your personal gateway is live

</div>
