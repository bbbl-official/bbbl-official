## ⚙️ **HOW IT WORKS - معماری کامل**

<div align="center">
  
  ```
  ┌─────────────────────────────────────────────────────────────────┐
  │                                                                 │
  │   ۱. کاربر فایل worker رو دانلود میکنه                          │
  │   ۲. آپلود روی Cloudflare Workers (رایگان)                     │
  │   ۳. دریافت لینک شخصی گیت‌وی                                    │
  │   ۴. هر سایتی رو تایپ میکنه → آزادانه میره 🎉                   │
  │                                                                 │
  └─────────────────────────────────────────────────────────────────┘
  ```
  
</div>

### 🔄 جریان ترافیک:

```
کاربر در ایران
      │
      ▼
[ISP] ← فقط ترافیک رمز شده میبینه ❌
      │
      ▼
[Cloudflare Edge - آلمان] ← اولین گیت‌وی
      │
      ▼
[BBBL Router] ← تشخیص سایت مقصد
      │
      ▼
[Cloudflare Edge - آمریکا] ← درخواست از خارج
      │
      ▼
[YouTube/Google/X] ← سایت مقصد
      │
      ▼
کاربر محتوا رو میبینه ✅
```

### 🛡️ لایه‌های امنیتی:

| لایه | فناوری | وظیفه |
|------|--------|-------|
| 1 | TLS 1.3 + ESNI | رمز کردن ترافیک |
| 2 | 300+ PoPs | رد شدن از 300 سرور جهانی |
| 3 | IP Rotation | آیپی جدید برای هر درخواست |
| 4 | Zero-Logging | هیچ لاگی ذخیره نمیشه |
